# Input data

Large raw sequencing files are not included in this repository.

Place GEO supplementary/raw files in these folders before running the analysis:

- `data/GSE238140/` — primary human fetal brain-derived neural tissue scRNA-seq dataset
- `data/GSE230571/` — contextual Vero/moDC scRNA-seq dataset
- `data/GSE97919/` — contextual bulk RNA-seq dataset

The scripts expect 10X-style sample directories for the scRNA-seq datasets and GSM-level count files or the `GSE97919_RAW.tar` archive for GSE97919.
