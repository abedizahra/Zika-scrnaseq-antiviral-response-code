# راهنمای آپلود این کدها در GitHub

## روش ساده از داخل سایت GitHub

1. وارد حساب GitHub شوید.
2. روی **New repository** بزنید.
3. نام پیشنهادی مخزن:
   `zika-scrnaseq-antiviral-response-code`
4. گزینه **Public** را انتخاب کنید تا داور بتواند کدها را ببیند.
5. گزینه‌های ساخت README یا .gitignore را فعال نکنید، چون این فایل‌ها از قبل آماده شده‌اند.
6. روی **Create repository** بزنید.
7. فایل ZIP آماده‌شده را در کامپیوتر خود extract کنید.
8. در صفحه مخزن GitHub، گزینه **uploading an existing file** را بزنید.
9. همه فایل‌ها و پوشه‌های داخل پوشه `zika-scrnaseq-antiviral-response-code` را drag/drop کنید.
10. پایین صفحه در قسمت commit message بنویسید:
    `Initial upload of analysis scripts`
11. روی **Commit changes** بزنید.
